# 🎯 Objetivo

Este repositório foi criado para armazenar os exercícios e atividades desenvolvidos ao longo da disciplina de Lógica de Programação, fortalecendo conceitos fundamentais utilizados no desenvolvimento de software.

---

# 🛠️ Ferramentas Utilizadas

- Portugol
- VisualG
- VS Code
- Git & GitHub
